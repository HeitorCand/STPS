STPS is a trust scoring platform for teams that operate Solana protocols. It
turns governance signals and operational risk into a protocol certificate that
can be read in the dashboard and consumed through an authenticated SDK surface.

## What STPS measures

The score belongs to the **protocol**, not to a wallet and not to a single
transaction. Transactions are input events that may change the trust state of a
protocol.

Examples of signals:

- timelock removed or set to zero;
- multisig threshold lowered;
- new or unknown signer added;
- emergency key usage;
- durable nonces and latent administrative permissions;
- asset legitimacy and liquidity signals.

## Usage flow

1. The operator signs in with a Solana wallet.
2. The operator adds a protocol program address to the workspace.
3. The dashboard shows only the protocols linked to that account.
4. The operator creates a persistent token to use with `stps-sdk`.
