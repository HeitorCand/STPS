STPS é uma plataforma de trust scoring para equipes que operam protocolos na
Solana. Ele transforma sinais de governança e risco operacional em um certificado
legível no dashboard e em uma superfície autenticada para integrações via SDK.

## O que o STPS mede

O score é do **protocolo**, não de uma wallet e não de uma transação isolada.
Transações são eventos de entrada que podem alterar o estado de confiança do
protocolo.

Exemplos de sinais:

- timelock removido ou zerado;
- threshold de multisig reduzido;
- signer novo ou desconhecido;
- uso de chave de emergência;
- durable nonces e permissões administrativas latentes;
- sinais de legitimidade de ativos e liquidez.

## Fluxo de uso

1. O operador entra com uma wallet Solana.
2. O operador faz claim do program address do protocolo.
3. O STPS verifica controle por upgrade authority ou signer/admin conhecido.
4. O dashboard passa a mostrar apenas protocolos vinculados àquela conta.
5. O operador cria um token persistente para usar o `stps-sdk`.
