## Constructor

```ts
const client = new StpsClient({
  token: process.env.STPS_API_TOKEN!,
  apiUrl: "https://your-scoring-engine.example.com",
  timeoutMs: 10_000,
});
```

| Option | Required | Description |
| --- | --- | --- |
| `token` | Yes | Persistent token issued by the dashboard. |
| `apiUrl` | No | Scoring Engine URL. Uses the official environment when omitted. |
| `timeoutMs` | No | Request timeout in ms. |
| `fetchFn` | No | Custom `fetch` implementation. |

## `getProfile()`

Returns the user and session or token context used for authentication.

```ts
const profile = await client.getProfile();
console.log(profile.user.primaryWalletAddress);
```

## `getProtocols()`

Lists only the protocols linked to the token account.

```ts
const { count, protocols } = await client.getProtocols();
```

## `getProtocol(protocolAddress)`

Fetches a specific protocol inside the account scope.

```ts
const protocol = await client.getProtocol("9Q3ze...");
```

Returns `404` if the protocol is not linked to the account.

## `getScore(protocolAddress)`

Returns only the score object for the protocol.

```ts
const score = await client.getScore("9Q3ze...");
console.log(score.currentScore, score.activeFlags);
```

## `getHistory(protocolAddress)`

Returns the score change history.

```ts
const history = await client.getHistory("9Q3ze...");
```

## `subscribeToAlerts(protocolAddress, onUpdate, options?)`

Polls and calls the callback when score or flags change.

```ts
const stop = client.subscribeToAlerts(
  "9Q3ze...",
  (protocol) => {
    console.log(protocol.protocol.currentScore, protocol.status);
  },
  { intervalMs: 5000 },
);

stop();
```
