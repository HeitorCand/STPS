Intro

STPS is a trust validation layer for critical Solana protocol operations.

Solana Trust Protocol Standard

STPS assigns a Trust Score from 0 to 100 to DeFi protocols by reading governance, asset legitimacy and latent permission signals before critical execution.

STPS exists because a blockchain transaction can be valid and still be unsafe. The protocol operates in that gap: validating risk, rule coherence and operational impact before a governance action, admin permission or critical transaction becomes irreversible.

MVP scope

  • Anchor program for protocol trust certificates.
  • TypeScript Scoring Engine for risk events and score calculation.
  • Indexer or event mock for Helius Webhook style inputs.
  • SDK for querying protocol scores.
  • Dashboard and docs for explaining certificate state, risk flags and score history.
